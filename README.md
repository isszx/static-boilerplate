# static-boilerplate

This boilerplate for creating static html sites

Init:
```
yarn
# or
npm i
```

For dev:

```
yarn dev
# or
npm run dev
```

| check (localhost:2000)[localhost:2000]

For dev with auto open browser:

```
yarn dev:open
# or
npm run dev:open
```

For build

```
yarn build
# or
npm run build
```
